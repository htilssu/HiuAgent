COMPONENT BREAKDOWN
═══════════════════════════════════════════════════════════════════════════════════

1. HEADER (.calendar-header)
   ├─ Left Section (.header-left)
   │  ├─ Icon Wrapper (.icon-wrapper) - Gradient purple box
   │  └─ Title & Subtitle
   └─ Right Section (.header-actions)
      ├─ Previous Month Button
      ├─ Today Button
      ├─ Next Month Button
      └─ Create Event Button

2. MONTH DISPLAY (.month-display)
   └─ Month & Year (Vietnamese format)

3. CALENDAR GRID (.calendar-grid)
   ├─ Weekday Headers (7 items) - Purple gradient
   └─ Day Cells (42 items, 6 weeks)
      ├─ Day Number
      └─ Events Container
         └─ Event Items (color-coded)

4. EVENT SIDEBAR (.event-sidebar)
   ├─ Header - Purple gradient
   │  ├─ Date Display
   │  └─ Close Button
   └─ Content
      └─ Event Cards
         ├─ Header (Icon + Time)
         ├─ Title
         ├─ Description
         └─ Footer (Edit/Delete buttons)
